# 周周冥想项目演示

这是一个用于展示周周冥想（WeeklyZen）项目的演示网站，采用现代化的设计语言和交互体验。

## 项目结构

```
ppt/
├── index.html      # 主入口文件
├── intro.html      # 项目介绍页面
├── features.html   # 核心功能展示
├── ai.html         # AI 创新特性
├── demo.html       # 产品演示页面
└── README.md       # 项目说明文档
```

## 特性

- 响应式设计，适配多种设备
- 现代化的动画效果
- 键盘导航支持
- 全屏显示功能
- 左侧导航菜单
- 优雅的页面切换效果

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 使用键盘方向键（← →）或点击导航点切换幻灯片
3. 点击左上角菜单图标打开/关闭导航菜单
4. 点击全屏图标进入/退出全屏模式

## 技术栈

- HTML5
- Tailwind CSS
- Font Awesome
- 原生 JavaScript
